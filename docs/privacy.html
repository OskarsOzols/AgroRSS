Privacy Policy

Last updated: April 7, 2026

Overview

Latvijas Lauksaimniecības Ziņu Agregators ("the Application") is an automated RSS feed aggregator that collects publicly available news from Latvian agriculture websites and Facebook pages. The Application generates a public RSS/XML feed for use with feed readers such as Feedly.

Data Collection

The Application does not collect, store, or process any personal data from users. Specifically:

  • No user accounts or login is required
  • No cookies are set or read
  • No analytics or tracking scripts are used
  • No personal information is collected, stored, or shared
  • No user activity is logged

Data Sources

The Application accesses only publicly available information from:

  • Public RSS feeds from Latvian agriculture organizations
  • Publicly accessible news pages of Latvian agriculture websites
  • Public posts from Facebook pages via the Facebook Graph API

Only the following data is extracted from public sources: article title, article URL, publication date, and a brief description or excerpt. No private, restricted, or user-specific data is accessed.

Facebook Data

The Application uses the Facebook Graph API to read publicly available posts from public Facebook pages of Latvian agricultural organizations. The Application:

  • Does not access any private user profiles or data
  • Does not require Facebook login from end users
  • Does not store Facebook user IDs or personal information
  • Only reads public page posts (message text, post date, permalink)
  • Uses data solely to generate a publicly available RSS news feed

Data Storage

The only output of the Application is a single publicly hosted XML file (feed.xml) containing aggregated news article metadata (titles, URLs, dates, descriptions) from public sources. No databases or private data stores are used.

Third-Party Services

The Application is hosted on GitHub Pages and uses GitHub Actions for automated execution. No other third-party services are used. Please refer to GitHub's Privacy Statement for information about GitHub's data practices.

This privacy policy may be updated from time to time. Changes will be reflected in the "Last updated" date above.
